Weather Information App
========================

Developer:
Nike Nsikak-Nelson

Project Description:
---------------------
This Weather Information App is a Java Swing GUI application
that retrieves real-time weather information using the
OpenWeatherMap API.

Features:
---------
- Real-time weather updates
- Temperature display
- Humidity display
- Wind speed display
- Weather condition icons
- Dynamic background changes
- Search history tracking
- Celsius and Fahrenheit conversion
- Error handling for invalid cities

Technologies Used:
------------------
- Java
- Java Swing
- OpenWeatherMap API
- JSON Library

How to Run:
------------
1. Open project in VS Code
2. Add json.jar to Referenced Libraries
3. Run WeatherApp.java
